# 海龜突破交易系統

技能名稱：海龜突破交易系統
學習總結：根據 Donchian Channel 價格通道突破來判斷趨勢,在突破點進場並使用止損管理風險,讓利潤持續擴大的完整交易系統。
推薦原因：海龜交易法最大優勢在於提供明確的進出場規則,同時強調風險控管,避免主觀判斷造成的交易偏差。系統化的操作讓交易決策更客觀理性。
核心規則：
1. 進場: 當價格突破20天最高點做多,突破20天最低點做空
2. 加碼: 每上漲0.5ATR可加倉一次,最多加倉4次
3. 止損: 設定2ATR作為每筆交易的最大損失
4. 出場: 當價格跌破10天最低點平倉多單,漲破10天最高點平倉空單
5. 部位: 單一商品風險不超過帳戶1%,總風險不超過6%
