# 外資猛虎三擊戰法

技能名稱：外資猛虎三擊戰法

學習總結：精選外資連3買且量價齊揚的個股，結合均線多頭確認強勢，抓住法人建倉初期的趨勢飆股。

推薦原因：
台股高度依賴外資動向，當外資大量買超又有投信跟進時最容易產生飆股。此策略特別適合科技股補漲及產業輪動初期的強勢股追蹤。

核心規則：
1. 進場條件
- 外資連續3個交易日買超且累積買超量超過1000張
- 5日均量較20日均量放大50%以上
- K線站上5日、10日、20日均線，且三條均線呈多頭排列
- 最近5日股價創20日新高

2. 濾網條件  
- 排除投信連續賣超個股
- 排除當日跳空超過3%的個股
- 排除近期內有特殊新聞事件的個股
- 成交量需維持在500張以上

3. 出場條件
- 獲利超過10%先出半
- 跌破10日均線即停損
- 外資轉為連續賣超2日則提前出場
- 單筆交易最大損失設定為6%
