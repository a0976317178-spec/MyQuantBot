# 噬血獵獅戰法-跌停逆襲術

技能名稱：噬血獵獅戰法-跌停逆襲術

學習總結：
專抓市場過度恐慌拋售後的強勢反彈，透過籌碼、量能、基本面三重過濾，精準狙擊機構法人回補時機。

推薦原因：
台股漲跌幅限制使恐慌性跌停常製造超賣，加上散戶習慣性殺盤，讓法人有機會低價承接。此策略特別適合產業基本面仍佳、遭遇突發利空的優質股。

核心規則：

1. 進場條件：
- 前日跌停，且成交量為20日均量2倍以上
- 開盤即現強勢打開(開盤價較跌停價高出1%以上)
- 外資、投信連續3日賣超後轉為買超
- 現股當沖比重低於25%(避開投機股)

2. 濾網條件：
- 排除財務問題股(負債比>70%、速動比<1)
- 排除近期遭金管會處分或公司重大負面消息
- 主力持股比例需維持40%以上
- 股價需在季線之上

3. 出場條件：
- 獲利20%立即出場
- 開盤後跌破前一小時最低點即停損
- 持股不超過3天，無論盈虧必須出場
- 大盤跌破月線立即停損
